#  Income Tax Calculator

A modern and responsive **Income Tax Calculator** built using **HTML, CSS, and JavaScript**. The application allows users to calculate their estimated income tax based on annual income with a clean, user-friendly interface.

##  Features

* Modern and responsive UI
* Real-time income tax calculation
* Effective tax rate display
* Post-tax income calculation
* Input validation and error handling
* Mobile-friendly design
* Fast client-side calculations (no data is stored)

##  Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

##  Project Structure

```
├── index.html
├── style.css
├── script.js
└── README.md
```

##  Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter your annual income.
4. View your estimated tax instantly.

##  Disclaimer

This calculator provides an estimate based on the implemented tax slabs and is intended for educational purposes only. It should not be considered official tax advice.
